This application demonstrates basic CRUD (Create, Read, Update, Delete) operations on a PostgreSQL database using Python and Psycopg 3. 
It manages a simple students table

Prequisites:
* Python
* PostgreSQL server
* pgAdmin Portal

Running application:
* open and run Assignment3.sql in pgAdmin to create table
* Insert data into table
* Open /python app/db_script.py to configure database credentials
* Run python script to execut functions
